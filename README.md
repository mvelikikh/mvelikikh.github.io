# Описание

Технические заметки.

## Установка

```bash
pip install .
```

Опции разработчика:

```bash
pip install .[dev]
```

## Запуск

```bash
mkdocs serve
```

## Развёртывание

Автоматическое при `push` в ветку `master` через [GitHub Actions](https://docs.github.com/ru/actions) процесс: [ссылка на workflow](.github/workflows/build_and_deploy.yml)
